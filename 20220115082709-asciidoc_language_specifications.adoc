== AsciiDoc言語とは

:about_asciidoc_url: {asciidoc_docs_url}/#about-asciidoc
link:{about_asciidoc_url}[
AsciiDoc言語^]はMarkup言語です。

類似のものとしては、MarkdownやreStructuredTextがあります。
表作成やfileの全体・1部の取り込み(`include:<file_path>[]`)ができます。

:compare_asciiDoc_to_markdown_url: {asciidoc_docs_url}/asciidoc-vs-markdown/
link:{compare_asciiDoc_to_markdown_url}[Markdownとの比較^]

== {asciidoc_lang_repository_link}は更新していない

:asciidoc_lang_url: https://projects.eclipse.org/projects/asciidoc.asciidoc-lang
link:{asciidoc_lang_url}[
AsciiDoc言語はEclipse FoundationのEclipse open source projectです。^]
Eclipse Foundationには、
{asciidoc_lang_repository_link}がAsciiDoc言語の仕様書のrepositoryと記載しています。
しかし、2022年1月現在、{asciidoc_lang_repository_link}は5月の更新が最後です。

=== AsciidoctorとAsciiDocの仕様

:asciidoctor_documentation_url: {asciidoctor_docs_url}/asciidoctor/latest
:what_is_asciidoctor_url: {asciidoctor_documentation_url}/#what-is-asciidoctor
.Asciidoctorとは
link:{what_is_asciidoctor_url}[
AsciiDoc言語をどのように解釈するかの参考実装です。^]

:about_asciidoc_documentation_url: {asciidoc_docs_url}/#about-this-documentation
link:{about_asciidoc_documentation_url}[
Asciidoctorの実装により、AsciiDocの言語仕様を定義します。
AsciiDoc言語の最初のVersion仕様が確定するまで、これは続きます。^]
そのため、AsciiDoc言語の全ての仕様要件をAsciidotcor実装しています。

link:{asciidoc_lang_url}[
AsciiDoc言語project^]がAsciiDoc言語のdocumentを公開するまで、
AsciiDoc言語の仕様はAsciidoctorのdocumentになります。


== AsciiDoc.pyも{asciidoc_lang_doc_repository_link}に合わせようとしている

.AsciiDocとは
AsciiDocの最初のプロジェクトです。
"AsciiDoc.pyの仕様" = "AsciiDoc言語の仕様" と私は前まで考えていました。

link:{asciidoc_py_loadmap_url}[
