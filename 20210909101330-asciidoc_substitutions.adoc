== souce block内のURLをlinkにする

source、listingやliteral bloce内のURLをlinkにするには `macros` を使用します。

link:{asciidoctor_subs_document_url}/apply-subs-to-blocks/#the-subs-attribute[
`macros` があると、macro処理をします。^]
macro処理内で、block内のURLは、URL macroになります。

[CAUTION]
====
Asciidocでは使えません。Asciidoctorでは使えます。
====

.Code
[source, Asciidoc]
------
[source, Lua, subs="macros"]
----
-- https://neovim.io/doc/user/lsp.html#lsp-api
-- https://neovim.io/doc/user/lsp.html#lsp-api[LSP API^]
-- https://neovim.io/doc/user/lsp.html#lsp-api[
   Neovim LSP API^]
vim.lsp.buf_request()
----
